2013 Warframe

As you're revisiting the earliest versions, some things to note:

  • Usage of a client patch (such as the Bootstrapper) is optional for versions before update 8 thanks to the -webserver argument.
  • There is an alternative to SpaceNinjaServer for U8 and below, that being Mekael's WFClassic.

Downloading M:

There are several ways to download Warframe's Steam depot at manifest version :

  • You can use mango like so: mango download-and-install 230411
    • The resulting installation will be stored in install/230411/. You may delete the depot folder after.
  • You can use DepotDownloader like so: DepotDownloader -app 230410 -depot 230411 -manifest -username <your steam username> -remember-password
    • This requires a Steam account with Warframe in its library.
    • Note that DepotDownloader may report errors; these are generally only temporary and it will try again.
  • You can use Steam's console with the following command: download_depot 230410 230411
    • This requires a Steam account with Warframe in its library.
    • The resulting installation will be stored in C:\Program Files (x86)\Steam\steamapps\content\app_230410\depot_230411 or equivalent. Once the download is finished, the console will tell you the exact path. Note that you'll want to clean out this folder for subsequent depot downloads to avoid corruption.

Downloading

  1. Download the base version (M:)
    • You must download this exact manifest version as the update patch only contains the data for updates to this specific base version
  2. Download the update patch
    • From Mega
    • Via BitTorrent v1 compatible software: Magnet
  3. Now you can drop the update patch on top of the base version, replacing the EXE in the process
